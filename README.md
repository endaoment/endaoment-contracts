## Available Scripts

In the project directory, you can run:

### `npm run build`

Builds all contracts in `/contracts` directory using `oz compile`.<br/>
Outputs `/artifcts`, `/build` and `/cache` to root directory

### `npm run test`

Launches the test runner.<br />
